## 🎯 Success Criteria

You'll know the feature is working when:

✅ Database migration runs successfully  
✅ Feature toggle appears in Settings  
✅ Teachers see "Fee Collection" in sidebar  
✅ Teachers can collect fees  
✅ Collections show as "pending"  
✅ Admin sees collections in Finance tab  
✅ Admin can confirm collections  
✅ Payments are auto-created  
✅ Student balances update  
✅ Rejection flow works
